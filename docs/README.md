## Quick Reference

**Build**: `docker-compose run --rm pimod pimod.sh tsOS-base.Pifile`

**Base**: Raspberry Pi OS Lite (Debian Trixie)  
**Tool**: [pimod](https://github.com/Nature40/pimod)  
**Architectures**: arm64, armhf
